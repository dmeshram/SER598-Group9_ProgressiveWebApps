Why are you here!

To be a smart coder and learn about progressive web apps.

Welcome to the Green Loop Tutorial

This tutorial will guide you through building Green Loop, an offline-first sustainability logger. You will learn modern PWA techniques including caching strategies, service workers, local storage with IndexedDB, and full-stack development using React and Express.

Learning Outcomes

By the end of this tutorial, you will be able to:

  • Explain PWA fundamentals and web standards.
  • Build an installable PWA with service workers and manifest.
  • Implement offline data storage using IndexedDB.
  • Create a Dockerized full-stack app (React + Express).
  • Deploy using CI/CD pipelines on GitHub Actions.
  • Analyze the strengths and challenges of PWA technology.
