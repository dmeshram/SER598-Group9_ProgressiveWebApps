Welcome

This tutorial explains Progressive Web Apps (PWAs) with concise, practical examples. A PWA is a web application that leverages standard web technologies and a few additional platform features to deliver reliable, fast, and engaging user experiences across devices.

Learning outcome: after completing the tutorial you should be able to create a manifest, understand service worker basics, and test PWA behavior using browser DevTools.

What you will learn

Foundations

Core concepts

The PWA model, progressive enhancement, and performance basics.

Installability

Web app manifest

Manifest fields, icons, and how installability works on different platforms.

Offline

Service workers & caching

Service worker lifecycle, caching strategies, and offline fallbacks.

Testing

Tools & quality

Lighthouse audits, Workbox, PWABuilder, and debugging with DevTools.
