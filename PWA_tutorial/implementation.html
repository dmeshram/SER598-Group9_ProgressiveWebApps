Overview of implementation

This project has two components:

  • Backend: Spring Boot (Maven)
  • Frontend: Node + npm (dev server)

Prerequisites

  • Java 11+ ‚Üí java -version
  • Maven 3.6+ ‚Üí mvn -v
  • Node.js 14+ ‚Üí node -v
  • npm ‚Üí npm -v

Run backend (Spring Boot)

mvn spring-boot:run
  • Run inside your backend folder if structured as /backend
  • Build JAR: mvn clean package
  • Run JAR: java -jar target/app.jar

Run frontend (Node / npm)

npm install
npm run dev
  • Dev port: usually 3000 or 5173
  • Production build: npm run build
